# Final Task - Jubelio 

## Prerequisite

● Install Ruby
`https://www.ruby-lang.org/en/documentation/installation/`

● Install Selenium Web driver 

● Add Gemfile on ruby
````
gem webdrivers
gem rspec
gem capybara
gem cucumber
````

● Install Git
`https://git-scm.com/downloads`
 
● Run command `bundle install`

● Initialize `cucumber --init`
